# 航变客服辅助 AI Demo

这是一个面向在线旅游平台团队游产品的航变客服辅助 MVP Demo。

## 功能特性

- **前置感知**：基于航班状态、延误分钟数、接送/核心项目缓冲时间触发预警
- **客户画像四维推断**：
  - 时间敏感度
  - 价格敏感度
  - 行程复杂度
  - 过往航变行为
- **方案生成**：自动生成三套方案
  - 稳妥保行程
  - 调整替代
  - 兜底补偿
- **最优路径排序**：结合时间、成本、敏感性和复杂度进行推荐排序
- **客服话术**：自动生成客服可复制话术
- **示例场景**：内置亲子演出延误、高价值客户航班取消、价格敏感客户轻微延误等场景

## 快速启动

### 环境要求

- Node.js 18+

### 本地运行

```bash
node server.js
```

默认访问地址：

```text
http://localhost:5173
```

如需指定端口：

```bash
PORT=5174 node server.js
```

> Windows PowerShell 可使用：
>
> ```powershell
> $env:PORT=5174
> node server.js
> ```
